# Nexus ROS

## Environments information
- OS version: Ubuntu 18.04
- ROS version: Melodic
- Board: Jetson Nano J47
- Prerequisite packages
  - rosserial

## Run guideline  
### Keyboard control
  1. Start ROS
  2. Start ROSSERIAL node
  3. Start keyboard_teleop node
  4. Start hardware_control node
